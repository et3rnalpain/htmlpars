В этой статье мы сделаем жизнь чуточку легче, написав легкий парсер сайта на python, разберемся с возникшими проблемами и узнаем <s>все муки пайтона</s> что-то новое.<br/>
<br/>
Статья ориентирована <u>на новичков</u>, таких же как и я.<br/>
<a name="habracut"></a><br/>
<h3>Начало</h3><br/>
Для начала разберем задачу. Взял я малоизвестный сайт новостей об Израиле, так как сам проживаю в этой стране, и хочется читать новости без рекламы и не интересных новостей. И так, имеется сайт, на котором постятся новости: есть новости помеченные красным, а есть обычные. Те что обычные — не представляют собой ничего интересного, а отмеченные красным являются самым соком. Рассмотрим наш сайт.<br/>
<br/>
<img src="https://habrastorage.org/r/w1560/webt/z9/-o/6m/z9-o6mxwrgi_6l9c-1kdpc82svu.png" data-src="https://habrastorage.org/webt/z9/-o/6m/z9-o6mxwrgi_6l9c-1kdpc82svu.png"/><br/>
<br/>
Как видно сайт достаточно большой и есть много ненужной информации, а ведь нам нужно использовать лишь контейнер новостей. Давайте использовать мобильную версию сайта, <br/>
чтобы сэкономить себе же время и силы.<br/>
<br/>
<img src="https://habrastorage.org/r/w1560/webt/c2/hl/et/c2hlettxhyfzvr1aveovj8rnqqs.png" data-src="https://habrastorage.org/webt/c2/hl/et/c2hlettxhyfzvr1aveovj8rnqqs.png"/><br/>
<br/>
Как видите, сервер отдал нам красивый контейнер новостей (которых, кстати, больше чем на основном сайте, что нам на руку) без рекламы и мусора. <br/>
<br/>
Давайте рассмотрим исходный код, чтобы понять с чем мы имеем дело.<br/>
<br/>
<img src="https://habrastorage.org/r/w1560/webt/pr/gr/gw/prgrgwsy7vj6elyqipmkpoc2pqk.png" data-src="https://habrastorage.org/webt/pr/gr/gw/prgrgwsy7vj6elyqipmkpoc2pqk.png"/><br/>
<br/>
Как видим каждая новость лежит по-отдельности в тэге 'a' и имеет класс 'lenta'. Если мы откроем тэг 'a', то заметим, что внутри есть тэг 'span', в котором находится класс 'time2', либо 'time2 time3', а также время публикации и после закрытия тэга мы наблюдаем сам текст новости.<br/>
<br/>
Что отличает важную новость от неважной? Тот самый класс 'time2' или 'time2 time3'. Новости помеченые 'time2 time3' и являются нашими красными новостями. Раз уж суть задачи понятна, перейдем к практике.<br/>
<br/>
<h3>Практика</h3><br/>
Для работы с парсерами умные люди придумали библиотеку «BeautifulSoup4», в которой есть еще очень много крутых и полезных функций, но об этом в следующий раз. Нам также понадобиться библиотека Requests позволяющая отправлять различные http-запросы. Идем их скачивать.<br/>
<br/>
(убедитесь, что стоит последняя версия pip) <br/>
<br/>
<pre><code class="bash">pip install beautifulsoup4 </code></pre><br/>
<pre><code class="bash">pip install requests</code></pre><br/>
Переходим в редактор кода и импортируем наши библиотеки:<br/>
<br/>
<pre><code class="python">from bs4 import BeautifulSoup
import requests</code></pre><br/>
Для начала сохраним наш URL в переменную:<br/>
<br/>
<pre><code class="python">url = 'http://mignews.com/mobile'</code></pre><br/>
Теперь отправим GET()-запрос на сайт и сохраним полученное в переменную 'page':<br/>
<br/>
<pre><code class="python">page = requests.get(url)</code></pre><br/>
Проверим подключение:<br/>
<br/>
<pre><code class="python">print(page.status_code)</code></pre><br/>
Код вернул нам статус код '200', значит это, что мы успешно подключены и все в полном порядке.<br/>
<br/>
Теперь создадим два списка (позже я объясню для чего они нужны):<br/>
<br/>
<pre><code class="python">
filteredNews = []
allNews = []
</code></pre><br/>
Самое время воспользоваться BeautifulSoup4 и скормить ему наш page, указав в кавычках как он нам поможет 'html.parcer':<br/>
<br/>
<pre><code class="python">soup = BeautifulSoup(page.text, "html.parser")</code></pre><br/>
Если попросить его показать, что он там сохранил:<br/>
<br/>
<pre><code class="python">print(soup)</code></pre><br/>
Нам вылезет весь html-код нашей страницы.<br/>
<br/>
Теперь воспользуемся функцией поиска в BeautifulSoup4:<br/>
<br/>
<pre><code class="python">allNews = soup.findAll('a', class_='lenta')</code></pre><br/>
Давайте разберём поподробнее, что мы тут написали.<br/>
<br/>
В ранее созданный список 'news' (к которому я обещал вернуться), сохраняем все с тэгом 'а' и классом 'news'. Если попросим вывести в консоль все, что он нашел, он покажет нам все новости, что были на странице:<br/>
<br/>
<img src="https://habrastorage.org/r/w1560/webt/b3/2p/zb/b32pzbm08z1qgoukldd_zaaryp8.png" data-src="https://habrastorage.org/webt/b3/2p/zb/b32pzbm08z1qgoukldd_zaaryp8.png"/><br/>
<br/>
Как видите, вместе с текстом новостей вывелись теги 'a', 'span', классы 'lenta' и 'time2', а также 'time2 time3', в общем все, что он нашел по нашим пожеланиям.<br/>
<br/>
Продолжим:<br/>
<br/>
<pre><code class="python">for data in allNews:
    if data.find('span', class_='time2 time3') is not None:
        filteredNews.append(data.text)</code></pre><br/>
Тут мы в цикле for перебираем весь наш список новостей. Если в новости мы находим тэг 'span' и класc 'time2 time3', то сохраняем текст из этой новости в новый список 'filteredNews'. <br/>
<br/>
Обратите внимание, что мы используем '.text', чтобы переформатировать строки в нашем списке из 'bs4.element.ResultSet', который использует BeautifulSoup для своих поисков, в обычный текст. <br/>
<br/>
Однажды я застрял на этой проблеме надолго в силу недопонимания работы форматов данных и неумения использовать debug, будьте осторожны. Таким образом теперь мы можем сохранять эти данные в новый список и использовать все методы списков, ведь теперь это обычный текст и, в общем, делать с ним, что нам захочется.<br/>
<br/>
Выведем наши данные:<br/>
<br/>
<pre><code class="python">
for data in filteredNews:
    print(data)
</code></pre><br/>
Вот что мы получаем:<br/>
<br/>
<img src="https://habrastorage.org/r/w1560/webt/2s/n9/z1/2sn9z1tijed9en18unxruwpjynq.png" data-src="https://habrastorage.org/webt/2s/n9/z1/2sn9z1tijed9en18unxruwpjynq.png"/><br/>
<br/>
Мы получаем время публикации и лишь интересные новости.<br/>
<br/>
Дальше можно построить бот в Телеге и выгружать туда эти новости, либо создать виджет на рабочий стол с актуальными новостями. В общем, можно придумать удобный для себя способ узнавать о новостях.<br/>
<br/>
